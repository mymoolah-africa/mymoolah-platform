## Issues Encountered
- `@capacitor/cli@8.3.4` requires Node 22 and failed on local Node 18, so the implementation was aligned to Capacitor 6.2.1 with `capacitor-secure-storage-plugin@0.10.0`.
- `./gradlew assembleDebug` failed locally because Java 24 is active and Gradle/AGP generated by Capacitor 6.2.1 requires a supported Android JDK, typically JDK 17.
- Codespaces initially had no Android SDK; after installing JDK 17, command-line tools, `platform-tools`, `platforms;android-34`, and `build-tools;34.0.0`, `./gradlew assembleDebug` passed.
- npm reports dependency vulnerabilities in the wallet frontend tree; no broad `npm audit fix` was run because that can introduce unrelated breaking changes.
